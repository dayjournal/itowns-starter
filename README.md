# itowns-starter

![README02](img/README02.png)

Start iTowns easily.  
- [iTowns v2.42.5](https://github.com/iTowns/itowns)  
- [TypeScript v5.9.3](https://www.typescriptlang.org)  
- [Vite v7.3.1](https://vitejs.dev)  
- node v24.4.1
- npm v11.4.2

<br>

## Usage

![README03](img/README03.png)

<br>

Install package
```bash
npm install
```

<br>

build
```bash
npm run build
```

<br>

dev
```bash
npm run dev
```
